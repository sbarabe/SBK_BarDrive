Detailed Description

template<typename DriverT>
class SBK_BarDrive< DriverT >

Unified interface combining SBK_BarMeter and optional animation support.

Provides a simplified high-level interface to manipulate LED bar meters and trigger predefined animations, including fills, pulses, signal tracking, and block effects. Requires SBK_BARDRIVE_WITH_ANIM to enable animation support.
